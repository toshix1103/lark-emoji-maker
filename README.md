# lark-emoji-maker

Lark用絵文字・アイコンメーカー

## 概要

このプロジェクトは `開発/projects/lark-emoji-maker/` に配置されています。

## セットアップ

```bash
cd projects/lark-emoji-maker
npm install
```

## 開発

```bash
npm run dev
```

## ビルド

```bash
npm run build
```
